Surgimento da internet

Voltar

ARPANET: Em 1969, a ARPANET (Advanced Research Projects Agency Network) foi lançada pelo Departamento de Defesa dos Estados Unidos. Foi a primeira rede a utilizar o protocolo TCP/IP (Transmission Control Protocol/Internet Protocol), que mais tarde se tornaria a espinha dorsal da Internet.

Desenvolvimento do TCP/IP: Em 1974, Vinton Cerf e Bob Kahn desenvolveram o TCP/IP, um conjunto de protocolos que permitia a comunicação entre diferentes redes de computadores. Este foi um avanço crucial para a criação de uma rede de redes, ou seja, a Internet.

Padronização: Nos anos seguintes, várias organizações, incluindo a Internet Engineering Task Force (IETF) e a Internet Architecture Board (IAB), trabalharam na padronização e desenvolvimento contínuo dos protocolos de Internet.

Expansão: Nas décadas de 1980 e 1990, a Internet expandiu-se rapidamente, com a adição de mais redes e o aumento do número de usuários. A criação da World Wide Web por Tim Berners-Lee em 1989 e sua subsequente popularização na década de 1990 foram um dos principais impulsionadores do crescimento da Internet.

Como funciona o HTML

Voltar

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Funciona como uma linguagem de marcação, o que significa que você usa tags HTML para estruturar o conteúdo de uma página da web. Aqui está uma visão geral básica de como o HTML funciona: Estrutura de uma página HTML: Uma página HTML geralmente começa com uma declaração para definir o tipo de documento HTML. Em seguida, o conteúdo da página é encapsulado em elementos que contêm dois elementos principais: (cabeçalho) e (corpo).

Cabeçalho (): O cabeçalho contém informações sobre o documento, como o título da página, meta informações (como palavras-chave e descrição para mecanismos de busca), links para arquivos CSS (Cascading Style Sheets), scripts JavaScript e outros recursos.

Corpo (): O corpo da página contém o conteúdo visível que os usuários veem no navegador da web. Ele pode incluir texto, imagens, vídeos, formulários, listas e outros elementos.

O que são tags

Voltar

A função principal de uma tag é definir o significado ou a estrutura do conteúdo que a envolve. As tags geralmente são delimitadas por caracteres especiais, como "<" e ">", e podem ocorrer em pares ou individualmente, dependendo da linguagem de marcação.

Em HTML, por exemplo, as tags são usadas para marcar partes específicas do conteúdo da página da web e informar ao navegador como renderizar esse conteúdo. As tags HTML são frequentemente usadas em pares, com uma tag de abertura e uma tag de fechamento, como

No XML, as tags são usadas para definir estruturas hierárquicas de dados e documentos. Elas também podem ser usadas para identificar elementos específicos em um documento XML.

Em resumo, uma tag é uma unidade de marcação usada para identificar e definir partes específicas de um documento ou arquivo de texto, e é uma parte fundamental de muitas linguagens de marcação usadas na criação de conteúdo para a web, armazenamento de dados estruturados e outros fins.
